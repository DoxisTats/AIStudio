# Doxis Business Studio — AI Agent Wizard Wireframes

High-fidelity HTML/CSS wireframes for the AI agent creation wizard in Doxis Business Studio.

## Screens

| File | Screen |
|------|--------|
| `screen-1-ai-manager.html` | AI Agent Manager — agent list overview |
| `screen-2-identity.html` | Step 1 · Identity — name, task description, technical configuration |
| `screen-3-data-access.html` | Step 2 · Permissions — data access and application permissions |
| `screen-4-mcp-tools.html` | Step 3 · MCP Integrations — DOCSIS MCP Server, Smart Bridges |
| `screen-5-context.html` | Step 4 · Context — document sources and knowledge base |
| `screen-6-activation.html` | Step 5 · Activation — trigger type, user groups, rate limits |
| `screen-7-review.html` | Step 6 · Review — summary before publishing |
| `screen-8-hr-onboarding.html` | Demo scenario · HR Onboarding agent |
| `screen-8-superhuman-search.html` | Demo scenario · Superhuman Search agent |
| `screen-5-skills.html` | Skills configuration (earlier iteration) |

## Usage

Open any `.html` file directly in a browser — no build step or server required. All assets are local under `data/`.

## Design system

Built on the [Untitled UI](https://www.untitledui.com/) design system tokens and icon library, styled to match the Doxis brand.
